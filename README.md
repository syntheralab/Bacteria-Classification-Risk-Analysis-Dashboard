# 🦠 Bacterial Risk Analysis

## Overview

This project analyzes a dataset of 199 bacterial species to understand their classification, habitats, and potential risks to humans. Through exploratory data analysis and visualization, the project identifies trends and patterns that can support a better understanding of bacterial diversity and public health concerns.

---

## Objectives

* Analyze the distribution of harmful and non-harmful bacteria.
* Identify the most common bacterial families.
* Explore the habitats where bacteria are frequently found.
* Determine which bacterial families contain the highest number of harmful species.
* Generate meaningful insights through data visualization.

---

## Dataset Information

The dataset contains the following attributes:

| Column            | Description                                          |
| ----------------- | ---------------------------------------------------- |
| Name              | Bacterial species name                               |
| Family            | Bacterial family classification                      |
| Where Found       | Common habitat or environment                        |
| Harmful to Humans | Indicates whether the bacterium is harmful to humans |

### Dataset Statistics

* Total Records: 199
* Missing Values: 0
* Data Type: Categorical/Text

## Key Findings

* The dataset contains a diverse range of bacterial families and habitats.
* Only a portion of the bacterial species are harmful to humans.
* Certain bacterial families have a higher concentration of harmful species.
* Soil and human-associated environments are among the most common bacterial habitats.
